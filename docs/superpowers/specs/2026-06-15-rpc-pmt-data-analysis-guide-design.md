# RPC-PMT 主分析说明文档设计

## 目标读者

使用 `RPC_PMT_DataAnalysis.C` 处理示波器波形、检查 ROOT 输出并提取探测器性能的实验数据分析人员。

## 文档结构

文档采用实验分析流程导向，重点说明输入约定、物理量定义、计算公式、时间游走修正、ROOT 输出结构、运行方法、结果解释和已知限制。代码函数仅作为算法定位参考，不逐行讲解。

## 覆盖范围

- CSV 数据和电压目录格式。
- 单位、极性、基线及有效性校验。
- 幅度、上升时间、FWHM、电荷、CFD 和固定阈值时间。
- PMT 高斯定时与插值回退。
- `PMT - RPC` 时间差符号。
- 高斯统计拟合和时间分辨率。
- TA/TQ 修正、训练集/验证集划分和三参数拟合模型。
- ROOT 目录、事件树、拟合参数树、修正汇总树和图形对象。
- 配置、运行、检查、常见异常和分析注意事项。
